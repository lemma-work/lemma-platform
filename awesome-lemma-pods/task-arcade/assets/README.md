# Task Arcade — assets

Images and media referenced by the pod's `README.md` live here, so they travel with the pod
when someone forks it.

**How to use:**
- Drop image files in this folder (e.g. `banner.png`, `world-screenshot.png`, `quartermaster.png`).
- Reference them from the README with a relative path: `![alt](./assets/banner.png)`.
- Keep files reasonably small (banners < ~500 KB) so the repo stays light.

**Suggested set for the README:**
- `banner.png` — the hero banner (the 3D world, wide)
- `loop.png` — the assigned → cleared → placed → approved → established strip
- `quartermaster.png` — the Command Post / agent chat
